Appium Client Libraries

These are the various language bindings which you can use to send the proper http commands to appium, or any Mobile Driver which implements the Mobile JSON Wire Protocol.

  • wd.js, on npm
  • java-client, Maven module
  • appium_lib, Ruby gem
  • Appium-Python-Client, on pypi
  • Appium C# Webdriver, on NuGet
